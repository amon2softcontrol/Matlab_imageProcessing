Representation and Description

Objective:
	To represent and describe information embeded in an image in other forms
that are more suitable than the image itself.

Benefits :
	Easier to understand
	Require fewer memory, faster to be processed
	More 'ready to be used'

What kind of information we can use ?
	Boundary, shape
	Region
	Texture
	Relation between regions
